# BetterDuckov-v1.10

A comprehensive quality-of-life mod for [Escape from Duckov] that enhances various gameplay mechanics and provides extensive customization options.

## Features

### Inventory Management

- **Item Weight Reduction**: Adjust item weight from 0.1x to 1x
- **Item Stack Multiplier**: Increase stack sizes from 1x to 200x
- **Backpack Capacity**:
  - Backpack size multiplier (1x-10x)
  - Additional backpack slots (0-500)
- **Storage Capacity**:
  - Storage size multiplier (1x-10x)
  - Additional storage slots (0-1000)

### Equipment & Gear

- **Extra Totem Slots**: Add 1-50 additional totem slots
- **Extra Weapon Attachment Slots**: Additional slots for weapon attachments
- **Totem & Accessory Effects**:
  - Removes all negative effects from totems and accessories
  - Doubles positive effects from totems and accessories

### Gameplay Mechanics

- **Durability System**:
  - Repair cost reduction (0%-100%)
  - Durability multiplier (1x-10x)
- **Action Speed**: Increase action speed (1x-10x) for using items and searching
- **Instant Level Up**: Immediate level progression
- **Talent System**: Removes all negative effects from talents

### Loot & Enemies

- **Extra Loot**: The container will dynamically generate additional loot
- **Enemy Multiplier**: Increase enemy spawns (1x-10x)
- **Auto Loot Settings**:
  - Auto-pickup bullets (with quality threshold 1-6)
  - Wishlist-only pickup mode
  - Auto-unload ammo from loot
  - Auto-loot range (5-50 units)
  - Auto-loot scan interval (0.5-5 seconds)
  - Loot message position customization

### Economy & Shops

- **Shop Inventory Multiplier**: (1x-20x)
- **Black Market Upgrade Multiplier**: (1x-20x)
- **Buy All Button**: Purchase entire shop inventories at once

  ### Enemy Health Scaling

  - **Scaling Factor**: (0.01x-100x)

## Installation & Usage

1. **First Time Setup**: Press `F8` or `Q` and `E` keys in the main game menu to open the mod settings interface
   - **Note**: You may need to press the key combination twice for the first time activation

2. **Important Warning**:
   - **REMOVE ALL TOTEMS AND ACCESSORIES BEFORE UPDATING THE MOD** to prevent item loss!

3. **Special Notes**:
   - Do not set aiming time to 0 - pressing ESC while aiming with zero aim time may cause a black screen
   - Additional equipment slots are hidden by default but support scroll wheel navigation
   - Compatible with game version 1.16 with no known issues

## Configuration

All features can be customized through the in-game mod settings menu. Adjust values to your preferred gameplay style.

## Installation (Non-Steam Windows)

1. Download the mod zip from Releases page
2. Navigate to your game's `Duckov_Data` folder
3. Create `Mods` folder (if missing)
4. Extract zip to `Mods` folder
5. Final path should contain all DLL files:
Duckov_Data/Mods/BetterDuckov_v1.10/

- bigInventory.dll
- 0Harmony.dll
- MonoMod.RuntimeDetour.dll
- MonoMod.Utils.dll

6.Launch game → Mods menu → Enable mod
